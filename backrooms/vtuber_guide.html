THIS IS THE DEFAULT METHOD

    First, to make this way easier, you should have a reference image of your character. I recommend using a photo of your character from somewhere like Pixiv or DeviantArt. You can also use a picture of a character from a game or anime. This will mostly help get the colors correct.
    
    Download and install Vroid Studio, you can also get this on steam for free. VRoid Studio
    It will be easiest to use this program like a video game character creation. There are also lots of assets on Booth.pm where some are paid and some are free. Whe you are done remember to save a .vroid file but also export a .vrm file. The .vrm file is the one you will need for the next step.

    Now you need a tool that will use motion capture.
        Free VSeeFace
          Requires a webcam.

        Free VMagicMirror
          Does not require a webcam.
