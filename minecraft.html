Server Rules

These rules apply to all server versions.
  1. Self Care - Always prioritize your real-life responsibilities over the game.
  2. Server Damage - Don't intentionally lag or break the server; this results in a ban.
  3. No Disrespect - Avoid excessive rudeness; players are here to relax.
  4. Player Consent - Obtain consent for actions like griefing, fighting, raiding, or sharing.
  5. Conversations - Avoid topics similar topics like sex, violence, or hate speech.
  6. Be Independent - Build independent settlements; team activities require player consent.
  7. About Hacks - Hacks are allowed unless they cause server damage.
  8. Vote Ban - The community can vote to ban a player if needed.
  9. Changes - Majority approval is required for changes to the server or the rules.
