About Vanity

Vanity has been a passion project since about 2017.
The project has never been serious and never will be.
The concept: "How ridiculous can a discord bot be?"

Historical features

features of vanity at the current version is using an uncensored llm based on llama3, I will refer to this model as lexi3 and if enabled will use a voice model to speak the text outloud (sent to discord as an mp3 file) Historiacal versions of vanity would allow her to connect to any vanilla minecraft server and play with the player with commands (although plans were made to procedurly understand what a player was doing and assist the player by "helping") I have always tried to add some sort of text recognition so in the far past I would have vanity recognize various forms of cussing and then overreact to the user by calling them names, remember this is in good fun.
