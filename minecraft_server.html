Global Server Rules

These rules apply to all server versions.

  • 1 | Self Care - Real life is more important than the game; please take care of yourself.
  • 2 | Player Consent - Consent is required for any in-game actions such as fighting, raiding, or sharing.
  • 3 | No Disrespect - Don't be excessively rude and disrespectful; players are here to relax.
  • 4 | Server Damage - Don't intentionally lag the server or otherwise try to break it; this will result in a global ban.
  • 5 | About Hacks - Hacks and exploits are not necessarily against the rules, but if they break rule 4, you will be stopped by force.
  • 6 | Be Independent - Players are encouraged to build independent settlements; see rule 2 for team activities.
  • 7 | Language - All members are English-speaking; please use only English.
  • 8 | Conversations - Don't insist on conversations that make players uncomfortable.
  • 9 | Vote Ban - If the community needs to vote to ban a player, they can, if necessary.
  • 10| Changes - Changes to the server, community, rules, or anything else need to be agreed upon by the majority of the community.

Report a Server Issue - If you encounter an issue with the server, please contact Admin Osyra. Provide a detailed description of the problem and include screenshots if necessary. Avoid vague reports such as "I have an issue with the server" without further details.
Report a Player Issue - If you have a problem with another player, reach out to any Staff member. Clearly explain the issue, including any relevant off-server interactions if applicable.
